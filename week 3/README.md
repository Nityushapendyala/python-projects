# Expense Tracker 💰

## About This Project

This project is a simple Expense Tracker built using Python. I created it to practice file handling, JSON storage, and working with numerical data while building a useful real-world application.

The application allows users to record expenses, organize them by category, search for expenses, and calculate total spending. All data is stored in a JSON file so records are preserved even after closing the program.

## Features

* Add expenses
* View all expense records
* Search expenses by category
* Calculate total expenses
* Store data using JSON
