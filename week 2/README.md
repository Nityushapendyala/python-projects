# Library Management System 📚

## About This Project

This project is a simple Library Management System built using Python. I created it as part of my backend learning journey to improve my understanding of file handling and working with JSON data.

The application runs through the command line and allows users to manage library books by adding, viewing, searching, issuing, and returning books. Unlike my first project, the data in this system is stored in a JSON file, so book records are saved even after the program is closed.

## Features

* Add new books to the library
* View all available books
* Search for a book using its Book ID
* Issue books
* Return books
* Store records permanently using JSON

## How to Run

1. Open the project folder.
2. Run the Python file:

```bash
python Lms.py
```

3. Follow the menu options displayed in the terminal.
