# Student Management System (CLI)

## About the Project

This is a simple Student Management System built using Python and operated through the command line. I created this project to strengthen my understanding of Python fundamentals such as functions, lists, dictionaries, loops, and conditional statements.

The application allows users to add, view, search, and delete student records through a menu-driven interface.

## Features

* Add new student records
* View all students
* Search for a student using their roll number
* Delete student records
* Simple and easy-to-use command-line interface

## Tech Stack

* Python 3

## How to Run

1. Clone this repository:

```bash
git clone <repository-url>
```

2. Open the project folder:

```bash
cd Student-Management-System
```

3. Run the program:

```bash
python Cli.py
```
